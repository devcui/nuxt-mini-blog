# Webpack 5 新特性与配置优化

Webpack 5 带来了许多激动人心的新特性，大幅提升了构建性能。

## Module Federation

模块联邦是 Webpack 5 最重要的新特性：

```javascript
const ModuleFederationPlugin = require('@module-federation/webpack');

module.exports = {
  plugins: [
    new ModuleFederationPlugin({
      name: 'host',
      remotes: {
        remote: 'remote@http://localhost:3001/remoteEntry.js'
      }
    })
  ]
};
```
