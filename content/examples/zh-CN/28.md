# MongoDB 聚合查询深入解析

MongoDB 的聚合框架提供了强大的数据处理能力。

## 聚合管道

聚合操作通过管道处理数据：

```javascript
db.orders.aggregate([
  { $match: { status: "completed" } },
  { $group: { 
    _id: "$customerId", 
    totalAmount: { $sum: "$amount" },
    orderCount: { $sum: 1 }
  }},
  { $sort: { totalAmount: -1 } },
  { $limit: 10 }
]);
```

## 性能优化

```javascript
// 使用索引优化聚合查询
db.orders.createIndex({ status: 1, customerId: 1 });
```
